# Zoomify
Allows you to zoom while on the pause menu

# Controls

## Desktop
- <cy>Scroll Wheel</c> to zoom in and out
- <cl>Home</c> to <cp>Toggle the Pause Menu</c> (Can be customized)

## Mobile
- Press the <cy>Magnifying Glass</c> button in the pause menu to enter zoom mode
- Pinch to zoom, use the back button in the top left to exit
